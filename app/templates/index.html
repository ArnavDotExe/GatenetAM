How it Works

1. Initialization: When you press Start Monitoring, the system clears any old data and connects to the image feed.

2. Live Detection: We process the stream frame-by-frame. An anomaly detection model scans each frame to locate potential defects (indicated by the cropped region).

3. Classification: If an anomaly is located, a secondary classification engine analyzes the crop to determine the exact type of error and our confidence level.

You can pause the stream at any time to inspect a specific frame, or restart the monitoring to begin fresh.
